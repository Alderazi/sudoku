How to Play Sudoku

Sudoku is a logic-based number puzzle game. Your goal is to fill the 9×9 grid so that each column, each row, and each of the 3×3 boxes contains all the numbers from 1 to 9.

Rules

  • Each row must have the numbers 1 to 9 with no repetition.
  • Each column must have the numbers 1 to 9 with no repetition.
  • Each 3×3 box must have the numbers 1 to 9 with no repetition.

How to Play

  1. Click a cell to select it.
  2. Choose a number from the number pad below (or type it with your keyboard).
  3. If it's correct, it stays. If it's wrong, you may get a mistake.
  4. You lose if you make 3 mistakes (or whatever your game's rule is).

Tips

  • Start with boxes, rows, or columns that have the most numbers already filled in.
  • Use logic—not guessing—to figure out where numbers go.
  • Use note mode if you're unsure about a number. It helps track possibilities.
